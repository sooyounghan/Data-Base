-----
### SELECT문
-----
1. 데이터베이스에 저장된 데이터를 가져오는 명령문
2. 상황이나 조건에 맞는 데이터를 추출 가능
3. 기본 구조

```sql
SELECT {* | [테이블명.]컬럼명 1, .. 컬럼명 [AS 별칭]} / 수식 또는 함수도 가능
FROM [데이터베이스명.]테이블명
[WHERE 조건]
[GROUP BY 그룹기준]
[HAVING 그룹조건]
[ORDER BY 정렬 기준 정렬 방법];
```

4. 예제
  A. 사원의 모든 정보를 가져옴 (사원정보 : employees 테이블)
```sql
SELECT *
FROM EMPLOYEES;
```
